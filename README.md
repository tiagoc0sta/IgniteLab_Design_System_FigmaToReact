
# Login Page  - ReactJS   

ReactJS - Implementing a Login Page from Figma to Raact - Design Systems.

 
Technologies used:

- ReactJS
- Typescript
- Phosphor-react 
- Tailwind
- React hook form 
- Postcss 
- Autoprefixer 
- Storybook
- Clsx 
- Radix UI 
- Axios 
- Mock service worker
